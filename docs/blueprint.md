# **App Name**: AirCube

## Core Features:

- Device Connection: Connects to ESP32 device via Bluetooth or Wi-Fi to fetch air quality data.
- Real-time Data Display: Display CO2 ppm, CO ppm, temperature, and humidity levels in real-time.
- Threshold alerts: Alert the user if the air quality data is outside a preset desired threshold.

## Style Guidelines:

- Primary color: Deep blue (#2E9AFE), reminiscent of clean air and technology, provides a calming yet modern feel.
- Background color: Light gray (#F5F5F5) offers a clean and unobtrusive backdrop, ensuring data visibility.
- Accent color: Electric blue (#7DF9FF) as highlights, drawing attention to real time numbers and alerts.
- Clean, sans-serif fonts for data clarity and readability.
- Use minimalist icons for air quality parameters, alerts, and connectivity status.
- A modular layout with clear sections for each data point, ensuring information is easily digestible at a glance.
